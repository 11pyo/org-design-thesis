## 연구 활동 / Working Papers

### 1. *Volatility Reverses the Horizontal-Organization Advantage* (2026)
- **저자:** 한원표 (단독)
- **분야:** 조직설계 / 조직경제학 / 계산사회과학
- **요약:** 환경 변동성이 조직구조의 수평 우위를 어떻게 반전시키는지 NK fitness landscape 시뮬레이션으로 검증.
- **규모:** 9개 사전등록 강건성 배터리에 걸친 36,900 회 시뮬레이션 실행
- **핵심 결과:** Spearman ρ(변동성, V−H 격차) = +1.00 / log(ω)에 대한 slope = +0.030 (95% CI [+0.026, +0.033], permutation p < 0.002). Burns-Stalker (1961) 60년 된 가설을 정량적으로 반증
- **공개 자료:** SSRN [6634219](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6634219) (review 단계) · 코드+데이터 CC BY 4.0 · Zenodo DOI [10.5281/zenodo.19707964](https://doi.org/10.5281/zenodo.19707964) · GitHub `11pyo/org-design-thesis`

### 2. *Adaptive Limits of Algorithmic Cryptocurrency Trading from a Complex Adaptive Systems Perspective* (2026)
- **저자:** 한원표 (단독)
- **분야:** 알고리즘 트레이딩 / 행동금융 / 강화학습
- **요약:** 복잡적응계 관점에서 암호화폐 알고리즘 트레이딩의 한계와 인간–AI 협업 모델 실증연구.
- **규모:** 10개 코인 × 45개월 × 6종 보완 분석 = 160+ 백테스트 실험
- **핵심 결과:** 단순한 실시간 EMA(50/200) regime detector가 always-on 알고리즘보다 유의하게 우수 (p = 0.036), 사후 정보(hindsight) 없이 인간–AI 협업 프레임워크 실증 근거 제시
- **공개 자료:** SSRN [6477100](https://papers.ssrn.com/sol3/papers.cfm?abstract_id=6477100) — **2026-04-28 정식 게재**
